## 1. 系统概述

### 1.1 系统简介

校园点餐管理系统是一个面向高校师生的在线订餐平台，提供便捷的校园周边美食订购服务。系统支持三种用户角色：

- **普通用户（学生/教职工）**: 浏览商家、下单购买、管理订单
- **商家用户**: 管理店铺、处理订单、查看经营数据
- **平台管理员**: 系统管理、用户管理、数据监控

### 1.2 系统特点

| 特性 | 说明 |
|------|------|
| 🚀 快速下单 | 简洁的下单流程，3步完成订餐 |
| 📱 响应式设计 | 支持手机、平板、电脑多端访问 |
| 💳 多种支付 | 支持支付宝、微信支付等多种方式 |
| 📍 地址管理 | 支持多个收货地址，快速选择 |
| 🔔 实时通知 | 订单状态变更实时推送 |
| 📊 数据可视化 | 商家端提供详细的数据统计 |

### 1.3 系统要求

**浏览器支持**:
- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+

**推荐分辨率**: 1920×1080 或更高（PC端）；375×812（移动端）

## 2. 快速开始

### 2.1 访问系统

打开浏览器，输入系统地址：`http://localhost:5173`（开发环境）

### 2.2 注册账号

1. 点击首页右上角「登录」按钮
2. 在登录页面点击「立即注册」
3. 填写注册信息：
   - 手机号（必填）：输入11位手机号码
   - 密码（必填）：6-20位字符，建议包含字母和数字
   - 验证码（必填）：点击获取验证码
4. 点击「注册」完成账号创建

### 2.3 登录系统

1. 进入登录页面
2. 输入手机号和密码
3. 点击「登录」按钮
4. 登录成功后自动跳转到首页

### 3.3 购物车管理

#### 3.3.1 查看购物车

**进入方式**:
- 点击底部导航「购物车」
- 或在商家页点击底部购物车栏

#### 3.3.2 购物车操作

| 操作 | 说明 |
|------|------|
| 修改数量 | 点击 +/- 按钮调整商品数量 |
| 删除商品 | 左滑商品项或点击删除图标 |
| 清空购物车 | 点击右上角「清空」按钮 |
| 选择商品 | 勾选/取消勾选商品 |

#### 3.3.3 购物车状态

- 底部导航购物车图标显示当前商品总数
- 超过99件显示「99+」

### 3.4 下单流程

#### 3.4.1 结算页面

**进入方式**: 购物车点击「去结算」

**填写信息**:

1. **选择收货地址**
   - 点击地址栏选择已有地址
   - 或点击「添加新地址」

2. **确认订单信息**
   - 查看商品清单
   - 确认商家信息
   - 查看配送费

3. **填写备注（可选）**
   - 输入特殊要求，如「不要辣」、「多放葱」等

4. **选择支付方式**
   - 支付宝
   - 微信支付
   - 余额支付

#### 3.4.2 支付页面

**支付流程**:

```
提交订单
   ↓
生成订单号
   ↓
跳转支付页面
   ↓
选择支付方式
   ↓
完成支付
   ↓
跳转订单详情
```

**注意事项**:
- 订单需在15分钟内完成支付
- 超时未支付订单自动取消
- 支付成功后不可修改订单

### 3.5 订单管理

#### 3.5.1 查看订单

**进入方式**: 点击底部导航「我的」→「我的订单」

**订单状态说明**:

| 状态 | 说明 | 可操作 |
|------|------|--------|
| 待支付 | 订单已创建，等待付款 | 去支付 / 取消 |
| 待接单 | 商家未确认订单 | 取消订单 |
| 制作中 | 商家已接单，正在制作 | 查看进度 |
| 待取餐 | 商品已制作完成 | 确认取餐 |
| 已完成 | 订单已完成 | 评价 / 再次购买 |
| 已取消 | 订单已取消 | 再次购买 |

#### 3.5.2 订单操作

| 操作 | 说明 |
|------|------|
| 查看详情 | 点击订单卡片查看详细信息 |
| 取消订单 | 在「待支付」或「待接单」状态可取消 |
| 申请退款 | 在「待接单」状态可申请退款 |
| 确认收货 | 收到商品后点击确认 |
| 评价订单 | 完成后可对商家和商品进行评价 |

### 3.6 地址管理

#### 3.6.1 添加地址

1. 进入「我的」→「地址管理」
2. 点击「添加新地址」
3. 填写地址信息：
   - 联系人姓名
   - 手机号码
   - 所在地区（省/市/区）
   - 详细地址（宿舍号/楼栋）
4. 可选择设为默认地址
5. 点击「保存」

#### 3.6.2 编辑/删除地址

- **编辑**: 点击地址项右侧「编辑」图标
- **删除**: 左滑地址项或点击「删除」
- **设为默认**: 点击「设为默认」按钮

### 3.7 个人中心

#### 3.7.1 功能入口

点击底部导航「我的」进入个人中心

#### 3.7.2 功能列表

| 功能 | 说明 |
|------|------|
| 我的订单 | 查看所有订单 |
| 地址管理 | 管理收货地址 |
| 个人信息 | 修改昵称、头像等 |
| 账户安全 | 修改密码、绑定手机 |
| 消息通知 | 查看系统通知 |
| 帮助中心 | 常见问题解答 |
| 关于我们 | 系统版本信息 |
| 退出登录 | 安全退出账号 |

### 4.3 订单管理

#### 4.3.1 订单列表

**功能说明**:
- 按状态筛选订单（全部/待接单/制作中/待取餐/已完成）
- 按时间范围筛选（今日/近7天/近30天）
- 搜索订单号或手机号

#### 4.3.2 订单处理流程

```
用户下单
   ↓
[待接单] ← 商家收到通知
   ↓
商家点击「接单」
   ↓
[制作中] ← 开始制作商品
   ↓
制作完成，点击「完成制作」
   ↓
[待取餐] ← 通知用户取餐
   ↓
用户取餐/配送完成
   ↓
[已完成]
```

#### 4.3.3 订单操作

| 状态 | 可执行操作 |
|------|------------|
| 待接单 | 接单 / 拒单 |
| 制作中 | 完成制作 |
| 待取餐 | 确认取餐 |
| 已完成 | 查看详情 / 查看评价 |

### 4.4 菜品管理

#### 4.4.1 菜品列表

**功能说明**:
- 查看所有菜品
- 按分类筛选
- 按状态筛选（上架/下架）

#### 4.4.2 添加菜品

1. 点击「添加菜品」按钮
2. 填写菜品信息：
   - 菜品名称（必填）
   - 菜品分类（必填）
   - 价格（必填）
   - 原价（选填，用于显示折扣）
   - 菜品图片（必填）
   - 菜品描述（选填）
   - 规格选项（选填，如大份/小份）
3. 点击「保存」

#### 4.4.3 编辑/删除菜品

- **编辑**: 点击菜品项「编辑」按钮，修改信息后保存
- **下架**: 点击「下架」按钮，菜品对用户不可见
- **删除**: 点击「删除」按钮，永久删除菜品

#### 4.4.4 库存管理

- 设置菜品库存数量
- 库存为0时自动下架

## 7. 常见问题解答

### 7.1 账号相关

**Q1: 忘记密码怎么办？**

A: 
1. 在登录页面点击「忘记密码」
2. 输入注册手机号
3. 获取短信验证码
4. 设置新密码
5. 使用新密码登录

**Q2: 如何修改手机号？**

A:
1. 进入「我的」→「账户安全」
2. 点击「修改手机号」
3. 验证原手机号
4. 输入新手机号并验证
5. 完成修改

**Q3: 一个手机号可以注册多个账号吗？**

A: 不可以，一个手机号只能绑定一个账号。

### 7.2 订单相关

**Q4: 下单后可以修改订单吗？**

A: 订单提交后无法修改。如需修改，可以取消订单后重新下单（需在商家接单前）。

**Q5: 订单超时未支付会怎样？**

A: 订单将在15分钟后自动取消，商品库存会释放。

**Q6: 如何申请退款？**

A:
- 商家未接单前：可直接取消订单，款项原路退回
- 商家已接单后：需联系商家协商退款

**Q7: 订单显示已完成但没收到商品？**

A: 
1. 先检查收货地址是否正确
2. 联系商家确认配送情况
3. 如确实未收到，可在订单详情申请售后

### 7.3 支付相关

**Q8: 支付失败怎么办？**

A:
1. 检查网络连接
2. 确认支付账户余额充足
3. 尝试更换支付方式
4. 如仍无法支付，请联系客服

**Q9: 支付成功后订单显示未支付？**

A: 可能是网络延迟导致，请等待1-2分钟后刷新页面。如仍显示未支付，请联系客服处理。

**Q10: 如何查看退款进度？**

A: 在「我的」→「我的订单」中找到对应订单，查看退款状态。退款一般会在1-3个工作日内到账。

### 7.4 商家相关

**Q11: 如何申请成为入驻商家？**

A:
1. 注册账号并登录
2. 进入「我的」→「商家入驻」
3. 填写商家信息并上传资质
4. 提交审核
5. 等待平台审核（1-3个工作日）

**Q12: 商家如何设置营业时间？**

A:
1. 进入商家端「店铺设置」
2. 点击「营业时间」
3. 设置每日营业时段
4. 保存设置

**Q13: 如何设置商品库存？**

A:
1. 进入商家端「菜品管理」
2. 点击商品「编辑」
3. 设置库存数量
4. 保存修改

### 7.5 技术相关

**Q14: 系统支持哪些浏览器？**

A: 支持 Chrome 90+、Firefox 88+、Safari 14+、Edge 90+ 等现代浏览器。

**Q15: 页面加载缓慢怎么办？**

A:
1. 检查网络连接
2. 清除浏览器缓存
3. 尝试刷新页面
4. 更换浏览器或设备

**Q16: 遇到系统错误怎么办？**

A:
1. 截图保存错误信息
2. 记录操作步骤
3. 联系客服并提供上述信息

## 8. 安全与隐私

### 8.1 账号安全

**密码安全建议**:
- 使用8位以上复杂密码
- 包含大小写字母、数字、特殊字符
- 定期更换密码（建议3个月）
- 不同平台使用不同密码

**登录安全**:
- 不要在公共设备上保存密码
- 登录后及时退出
- 发现异常登录立即修改密码

### 8.2 隐私保护

**我们收集的信息**:
- 账号信息（手机号、昵称）
- 订单信息（收货地址、购买记录）
- 设备信息（用于优化体验）

**信息使用**:
- 仅用于提供服务
- 不会出售给第三方
- 严格的数据加密存储

**隐私设置**:
- 可在「设置」→「隐私」中管理
- 可选择关闭个性化推荐
- 可申请导出或删除个人数据
